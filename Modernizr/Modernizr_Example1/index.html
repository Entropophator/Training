If you check the source of this page, you will see that "beer.js" is not loaded with the other scripts in the page head.
The javascript alert comes from the "beer.js" file, which represents a capability not supported by the browser as detected by Modernizr.

"loadchecks.js" calls Modernizr to check if "beer" is supported by the browser, as far as I am aware, no current browser supports any "beer" method, but they should.

If you check the debugger, in the html class tag of index.html, you will see a series of class="js no-....", this is Modernizr posting what methods
the browser page does NOT support, but should if it were HTML5 compliant. In loadchecks.js, I ask Modernizr if "beer" is supported, it is not,
so in loadchecks.js I tell Modernizr to load the "shim" file beer.js.
